
# Capstone Two: TF-IDF vs Transformer Sentiment Analysis

## Overview
This project compares traditional TF-IDF-based machine learning models with transformer-based NLP models for sentiment analysis.

## Key Findings
- Transformer models outperform TF-IDF models in accuracy and robustness.
- TF-IDF models remain useful for low-cost baseline solutions.
